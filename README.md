# AI-Native Architect Curriculum

AI가 구현 작업의 상당 부분을 담당하는 시대에, 제품·소프트웨어·데이터·AI·운영·비즈니스·조직을 통합적으로 설계하는 **AI-Native Product Systems Architect**로 성장하기 위한 5~10년 커리큘럼 설계 프로젝트.

## 구조

- [`prompts/curriculum-design-prompt-v2.md`](prompts/curriculum-design-prompt-v2.md) — 커리큘럼 설계 프롬프트 v2.2
- [`prompts/learner-input-template.md`](prompts/learner-input-template.md) — 학습자 입력·분기 갱신 보고 템플릿
- [`assessment/diagnostic-battery-v1.md`](assessment/diagnostic-battery-v1.md) — 시작 전 5세션 진단 배터리 (형식·루브릭, 문항은 세션 시 즉석 출제, 결과는 로컬 전용)
  - 12개 역량 도메인 체계 (CS 기초 ~ 메타학습)
  - 역량 깊이 모델 (L0~L4) 및 투자 방식 (Deep Core ~ Delegate)
  - 5단계 성장 모델 (Foundation Builder → AI-Native Meta Engineer)
  - 장기 앵커 제품 기반 프로젝트 커리큘럼
  - 인간·AI 역할 분담 프레임워크
  - 평가 체계 및 학습 운영 시스템

## 사용법

프롬프트를 LLM에 입력하고 `## 2. 학습자 입력과 진단` 섹션의 `[입력]` 항목을 본인 상황으로 채워 개인화된 커리큘럼을 생성한다. 출력은 3개 패스로 진행된다:

1. **Pass 1** — Master Architecture (전체 지도·전략)
2. **Pass 2** — First 12 Months Operating Plan
3. **Pass 3** — Stage Deep Dive (단계별 상세화)
